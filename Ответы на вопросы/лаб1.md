1. Пропуски данных

    Методы:

    - удаление строк/столбцов (если мало пропусков)
    - заполнение средним / медианой / модой
    - заполнение константой
    - KNN-imputation (реже)

    Суть:

    цель — сделать данные полными для обучения модели без потери информации или с минимальными искажениями

2. Масштабирование признаков

    Методы:

    - StandardScaler (среднее = 0, std = 1)
    - MinMaxScaler (0–1)
    - RobustScaler (через медиану)

    Зачем:

    - привести признаки к одному диапазону
    - убрать влияние “больших чисел”
    - улучшить работу градиентных и distance-based моделей
    - ускорить обучение

3. Алгоритмы, не зависящие от масштаба
    - Decision Tree
    - Random Forest
    - Gradient Boosting (XGBoost, LightGBM)

4. Приведение признаков к одному масштабу
    - стандартизация
    - нормализация
    - robust scaling

5. Снижение ошибки / обобщение
    - регуляризация (L1/L2)
    - кросс-валидация
    - feature selection
    - уменьшение переобучения
    - ансамбли
